# A Simple QQ Robot
## How to use:  

### Requirements: 

[Nodejs](nodejs.org)  
[Yarn](yarnpkg.com)

### Install

```sh
git clone https://github.com/masnn/qqbot.git
cd qqbot
yarn
yarn build
```
 
### Run CoolQ
[cqhttp](cqhttp.cc) is required for CoolQ, and it should run at websocket mode.

### Write config file
rename config.sample.json to config.json and make some change.

### Run bot  

```sh
node bin/robot.js
```
